# PerfViewSharp 可视化与交互指南 (Track 02)

欢迎使用 PerfViewSharp。本指南旨在帮助您快速掌握这款基于 GPU 驱动的高性能性能分析器。我们致力于提供一种“直觉化”的分析体验，让寻找瓶颈变得像翻阅地图一样简单。

## 1. 核心导航交互

PerfViewSharp 针对 **macOS 触控板** 和 **高精度鼠标** 进行了深度优化。

-   **平滑缩放 (Zoom)**：
    *   **操作**：双指在触控板上向上/向下滚动，或使用鼠标滚轮。
    *   **特性**：**以鼠标指针为中心**。缩放时，鼠标指向的函数块会保持在原位，方便您精准地“扎进”某段特定的执行细节。
-   **自由平移 (Pan)**：
    *   **操作**：左键点击并按住主视图任意位置，随后移动。
    *   **特性**：低延迟、零迟滞的拖拽体验，支持在漫长的时间轴中快速穿梭。
-   **全景导航 (Minimap)**：
    *   **操作**：点击底部鹰眼图中的任意区域，或拖拽其中的“白框”。
    *   **价值**：鹰眼图展示了全局数据的“热度分布”。白框实时代表您当前视口在全局中的位置，确保您在大尺度分析时永不迷路。

## 2. 智能可视化元素

-   **时间标尺 (Time Ruler)**：
    *   位于视图顶部。刻度会根据您的缩放级别自动切换颗粒度（从微秒到分钟），为您提供精准的时间空间参考。
-   **语义化染色 (Semantic Coloring)**：
    *   系统会自动根据方法名生成稳定的配色。
    *   **高亮指示**：选中的块显示为**黄色边框**，搜索匹配的块保持高亮度，未匹配块自动淡出。
-   **关键路径高亮 (Critical Path)**：
    *   带有**橙色外发光**的方块是系统自动识别出的耗时大户。这些通常是您调优的第一站。

## 3. 实时分析面板

-   **视口感知热点 (Viewport-Aware Hotspots)**：
    *   位于右侧面板。它会根据您**当前屏幕里看到的内容**实时重新计算 Top 5 最耗时的方法。
    *   **技巧**：当您平移视图时，观察此列表的变化，可以快速定位导致特定时间段卡顿的元凶。
-   **统计汇总**：
    *   点击任意方块，右侧将显示该方法在**整个追踪周期内**的总耗时、调用次数及其占据的 CPU 百分比进度条。

## 4. 源码闭环

-   **定位源代码**：
    *   **操作**：在火焰图块上**双击**，或点击右侧面板的“定位源代码”按钮。
    *   **效果**：系统将尝试自动打开您的 IDE（如 VS Code）并直接跳转到该函数所在的行号。
